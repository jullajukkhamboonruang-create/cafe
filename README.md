# ☕ ระบบ POS ร้านกาแฟและขนมเบเกอรี่ พร้อมระบบสะสมแต้ม (Cafe POS & Loyalty System)

แอปพลิเคชันระบบขายหน้าร้าน (POS) คาเฟ่ ดีไซน์ Light Mode สไตล์มินิมอลอบอุ่น มีระบบสะสมคะแนนสมาชิก คลังจัดการสินค้า หน้าจอวิเคราะห์ยอดขาย (Dashboard) เชื่อมต่อฐานข้อมูล Supabase และพร้อมเผยแพร่ขึ้น **Vercel** และจัดเก็บโค้ดบน **GitHub**

---

## 🌟 ฟีเจอร์เด่น
- **ขายหน้าร้าน (POS)**: ตะกร้าสินค้าอัปเดตแบบเรียลไทม์ ค้นหา/สะสมแต้มให้ลูกค้า แลกคะแนนเป็นส่วนลด (เช่น 10, 50, 100 แต้ม)
- **สแกนชำระเงินจำลอง**: รองรับการจ่ายเงินสด บัตรเครดิต และแสดงคิวอาร์โค้ดพร้อมเพย์ (PromptPay QR) จำลองตามยอดจ่ายจริง
- **แดชบอร์ดสรุปสถิติ (Dashboard)**: แสดงยอดขายวันนี้ จำนวนบิล สัดส่วนการขายของกาแฟและเบเกอรี่ และกราฟแนวโน้มยอดขายรายสัปดาห์ (Chart.js)
- **จัดการสินค้า (Catalog CRUD)**: เพิ่ม ลบ แก้ไขรายละเอียดสินค้า เปลี่ยนราคา และเปิด/ปิดการขายสินค้า
- **ระบบความปลอดภัยของพนักงาน (Auth)**: แยกพนักงานแต่ละคน และแอดมินผู้จัดการคลังสินค้า ป้องกันข้อมูลธุรกรรม
- **สถาปัตยกรรมไร้รอยต่อ (Hybrid Mode)**: 
  - **โหมดออนไลน์ (Vercel)**: ดึงรหัสเชื่อมต่อ Supabase ผ่านระบบ Vercel Environment Variables ที่ปลอดภัยสูงสุด
  - **โหมดออฟไลน์ (Local/Double-click)**: หากเปิดไฟล์ `index.html` บนเบราว์เซอร์โดยตรง ระบบจะสลับไปใช้คีย์จาก `localStorage` ที่พนักงานป้อนเองทันที

## 🚀 ขั้นตอนการจัดเก็บบน GitHub และเผยแพร่บน Vercel

### ขั้นตอนที่ 1: ติดตั้งฐานข้อมูล Supabase
1. สร้างโครงการบนเว็บ [Supabase](https://supabase.com)
2. ไปที่เมนู **SQL Editor** กด **New query** คัดลอกโค้ดทั้งหมดในไฟล์ `schema.sql` ไปวางแล้วกด **Run**
3. ไปที่เมนู **Settings (ฟันเฟือง) > API** เพื่อนำค่า **Project URL** และ **Anon API Key** มาเตรียมไว้

### ขั้นตอนที่ 2: อัปโหลดโค้ดขึ้น GitHub
หากคุณยังไม่มี Git ให้ลงทะเบียนใช้งานที่ [GitHub](https://github.com) จากนั้นรันคำสั่งเหล่านี้ใน Terminal ของคอมพิวเตอร์คุณเพื่ออัปโหลดโค้ด:

```bash
# 1. ริเริ่ม Git ในโฟลเดอร์นี้
git init

# 2. เพิ่มไฟล์ทั้งหมดเตรียมบันทึก
git add .

# 3. บันทึกประวัติการเปลี่ยนแปลง
git commit -m "เริ่มต้นสร้างระบบ POS ร้านกาแฟพร้อมระบบสะสมแต้ม"

# 4. สร้าง Repository บน GitHub จากนั้นคัดลอก Link มาและผูกความสัมพันธ์
git remote add origin https://github.com/ชื่อผู้ใช้ของคุณ/ชื่อเรโปของคุณ.git

# 5. อัปโหลดโค้ดขึ้น GitHub
git branch -M main
git push -u origin main
```

### ขั้นตอนที่ 3: เผยแพร่ขึ้น Vercel (Deploy to Vercel)
1. ลงชื่อเข้าใช้งานเว็บ [Vercel](https://vercel.com) (เข้าผ่านบัญชี GitHub ได้เลย)
2. คลิกปุ่ม **Add New > Project**
3. นำเข้า (Import) Repository ที่อัปโหลดไว้บน GitHub ในขั้นตอนที่ 2
4. **ตั้งค่าตัวแปรสภาพแวดล้อม (Environment Variables)**: ก่อนกด Deploy ให้คลี่แถบ "Environment Variables" ออกมา และป้อนค่าต่อไปนี้:
   - ชื่อตัวแปร: `SUPABASE_URL` | ค่าตัวแปร: *(ใส่ Project URL ของคุณจาก Supabase)*
   - ชื่อตัวแปร: `SUPABASE_ANON_KEY` | ค่าตัวแปร: *(ใส่ Anon Key ของคุณจาก Supabase)*
5. กดปุ่ม **Deploy**
6. ระบบจะดำเนินการรันแอป POS ร้านกาแฟและสร้างลิงก์เว็บไซต์ออนไลน์ให้คุณทันที! โดยพนักงานของคุณสามารถเปิดใช้งานผ่านอุปกรณ์ใดก็ได้โดยไม่ต้องตั้งค่าคีย์อีกรอบ
